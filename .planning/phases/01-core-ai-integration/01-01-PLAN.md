# Plan 01-01: Project Scaffold, Provider Abstraction, and Async Foundation

## Objective

Bootstrap the Homey AI Hub app project and establish the architectural foundation: project structure, abstract provider interface, and test infrastructure. No provider implementations yet — just the skeleton everything else builds on.

## Context

This is a Homey Python app targeting Homey Pro. The Homey CLI manages the project scaffold. Key constraints:
- Dependencies installed via `homey app dependency install <pkg>` (NOT pip) — this writes to `.homeycompose/` and pre-compiles into `.python_cache`
- `app.json` is auto-generated from `.homeycompose/app.json` — NEVER edit `app.json` directly
- Python 3.14 runtime (Homey platform pinned)
- Dev dependencies (pytest, ruff, pyright, homey-stubs) are installed via `pip` locally — they are NOT packaged

The working directory is `/Users/michelhelsdingen/Documents/Homey/homey-ai-hub`.

**Architecture confirmed (from research):**
- No drivers or devices — this is a service app using app-level Flow cards
- `app.py` exports `homey_export = App` (the App class)
- `lib/providers/base.py` defines the `LLMProvider` ABC (no Homey SDK dependency)
- The ABC interface: `async chat(messages, model, timeout) -> str`, `async list_models() -> list[str]`, `async test_connection() -> tuple[bool, str]`

    Create `lib/providers/base.py`:
    ```python
    """Abstract base class for all LLM providers."""
    from abc import ABC, abstractmethod


    class LLMProvider(ABC):
        """Common interface for all AI provider implementations.

        Providers are pure Python — no Homey SDK dependency.
        All methods are async to support non-blocking HTTP calls.
        """

        @abstractmethod
        async def chat(
            self,
            messages: list[dict],
            model: str,
            timeout: float | None = None,
        ) -> str:
            """Send messages to the AI and return the assistant response text.

            Args:
                messages: OpenAI-compatible message list, e.g.
                    [{"role": "user", "content": "Hello"}]
                model: Model ID string (provider-specific)
                timeout: Optional per-call timeout override in seconds.
                    If None, use the provider's default timeout.

            Returns:
                Assistant response as plain text string.
                On error, returns a human-readable error string starting with "Error:".
            """
            ...

        @abstractmethod
        async def list_models(self) -> list[str]:
            """Return available model IDs for this provider.

            Returns:
                List of model ID strings. Empty list on error.
            """
            ...

        @abstractmethod
        async def test_connection(self) -> tuple[bool, str]:
            """Test connectivity to the provider.

            Returns:
                (True, success_message) on success.
                (False, error_message) on failure.
            """
            ...
    ```

    Create `tests/__init__.py` as an empty file.

    Create `tests/conftest.py`:

    ```python
        homey.settings.set = AsyncMock()
        homey.settings.unset = AsyncMock()

        """Mock Homey FlowCardAction."""
        card = MagicMock()
        card.register_run_listener = MagicMock()
        card.register_argument_autocomplete_listener = MagicMock()
        return card
    ```
